# Carousel Copy Rules

**Important**: Read `resources/brand.md` first for language, tone, word bans, and CTA templates.

## Slide Hierarchy

### Slide 1 — THE Hook (90% of the work)
This slide determines whether someone swipes. It must stop the scroll.

**Rules:**
- Use marketing-psychology skill to identify the psychological lever (if installed, otherwise apply basic persuasion principles directly)
- Use copywriting skill to craft the hook — iterate until validated (if installed, otherwise write directly using the formulas below)
- Max 15 words on screen
- One clear, bold statement — no explanation yet
- Accent word in accent font italic (from brand.md)
- 1 word max highlighted in accent color (from brand.md)
- Must work as a standalone image (no context needed)

**Hook formulas that work:**
- Contrarian take: "X is dead. Long live Y."
- Shocking number: "+$500/month in AI tools"
- Identity shift: "You're not a [old] you're a [new]"
- Curiosity gap: "What nobody tells you about X"
- Direct challenge: "If you're still doing X, you're losing Y"

### Slide 2 — Second Hook
Instagram may show slide 2 on re-impression when someone scrolls past the carousel. It's a second chance.

**Rules:**
- Must function as standalone hook OR direct continuation of slide 1
- Never a throwaway transition slide
- Can be: the "why this matters" behind slide 1's claim
- Still punchy — max 20 words on screen
- Section marker at top ("01 — THE SHIFT" etc.)

### Slides 3-8 — Content
Each slide = one idea. No walls of text.

**Rules:**
- One concept per slide
- Max 25 words on screen
- Short sentences, active voice
- Use the address form specified in brand.md
- Each slide should make the reader want to see the next one
- Use section markers to create progression ("02 — THE SYSTEM", "03 — THE METHOD")
- Accent font italic on 1-2 key words max per slide
- Accent color highlight on 1 word max per slide

### End Slide — CTA
Always the last slide. Two options:

**Option A — Save:**
Use the Save CTA text from brand.md, with bookmark icon
- Use when the content is educational/tutorial
- Drives saves (algorithmic boost)

**Option B — Comment:**
Use the Comment CTA text from brand.md + description of what they get
- Use when you have a lead magnet or resource to give
- Keyword in accent color
- Description in italic (use Comment description from brand.md)

## Tone & Language

### Always
- Use the address form and tone specified in brand.md
- Direct, concise, active voice
- Write in the language specified in brand.md
- Use your audience's pain points (from brand.md or context/) as raw material for hooks

### Never
- AI-sounding negation patterns ("It's not X, it's Y")
- Hedging ("maybe", "potentially", "might")
- Emojis in the carousel itself (save for caption)
- See word bans in brand.md for brand-specific exclusions

## Content Level (TOFU/MOFU/BOFU)
Always identify before writing:

| Level | % | Purpose | Copy style |
|-------|---|---------|------------|
| TOFU | 50% | Attract audience with broad value | Tutorial, how-to, tool showcase — always framed for your audience |
| MOFU | 30% | Divide opinion, build trust | Hot takes, comparisons, tier lists, polarizing |
| BOFU | 20% | Destroy limiting beliefs, show proof | Client results, belief deconstruction, direct offer |
